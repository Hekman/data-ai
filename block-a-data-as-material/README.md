# Block A — Data as Design Material

**Block question:** *What does this data actually let me know, and what stories can I tell?*
Treats data as a designer's material — something you source, question, shape and compose with.
Targets **PLO 5** (Data Science Workflow & Analytical Reasoning) at *Advanced Beginner*.
